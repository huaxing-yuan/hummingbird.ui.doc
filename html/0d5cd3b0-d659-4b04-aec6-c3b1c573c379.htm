Introduction of Hummingbird UI Framework

Modern UI & MVVM

Hummingbird UI Framework is based on .NET Framework 4.5 and Windows Presentation Foundation (WPF), it provides highly customizable, and friendly User Interfaces. Contrary to traditional menu / content method, Hummingbird UI is based on the MVVM, it helps to separate different Views into different classes, instead of implementing menu code behinds and make much sub windows.

Hummingbird UI Framework restyles nearly all standard windows controls, just use standard windows controls, the styles are go with you.

As an example of Modern Window showed below, the main window is separated in several parts:

Global parts

These parts including

è Window Title : Icon, Application Name, Control Box

è Back Stage, Main Menu, Quick Access Button

è Taskbar replaces the traditional status bar

Global parts organizes windows and different functions provided by the application.

Some of the part is customizable by the View, such as Quick Access Buttons and Taskbar

View part

The actual content of the current selected menu item.

When using Hummingbird UI, you do not need to control the main menu selection and load/unload the content. The UI framework do it for you.

Themes and color schemes

Hummingbird UI Framework provides out-of-box 6 color schemes for Light backgrounded theme, and 4 color schemes for dark backgrounded theme.

And based on these color schemes, you can customize for both light and dark themes.
